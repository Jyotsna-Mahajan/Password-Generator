# 🔐 Python Password Generator

## 📌 Project Description

This project is a **Password Generator built using Python**.
It generates a **random and secure password** based on user preferences such as including **numbers and symbols**.

The program also checks the **strength of the generated password** and classifies it as **Weak, Medium, or Strong**.


## ⚙️ Features

* Generate random passwords
* User can choose password length
* Option to include **numbers**
* Option to include **symbols**
* Password strength checker
* Ensures password contains required characters
* Randomly shuffles characters for better security
